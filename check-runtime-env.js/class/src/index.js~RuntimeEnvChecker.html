    RuntimeEnvChecker

    Checker for Runtime Environment properties.

    Note that all methods here are static, so no need to instance this class; see it as an Utility/Companion class.

    Test:

    • ensure objects exported by index script, exists and are of the right type
    • ensure process info returns right values
    • ensure version checks are done in the right way
