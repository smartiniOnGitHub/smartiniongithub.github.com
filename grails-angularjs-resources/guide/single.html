1 Introduction

Grails-AngularJS-Resources

Introduction

This plugin makes it possible to use AngularJS in a Grails webapp.

All using Grails Resources plugin.

2 Usage

Installing the plugin

First you need to install the plugin into a Grails application: using the recommended way, configure it as a dependency in BuildConfig.groovy: 

plugins {
    runtime ':angularjs-resources:<plugin version>'
}

Requirements

  • a Servlet container to run the Web application

Sample usage

There is a sample/minimal test webapp (under test/webapp) with some examples on how to use the plugin, see sources.

4 Troubleshooting

Tricks

  • Module "index" missing in webapp, when trying to use angularjs.Nge2eController
    • Remember to define an "index" module in ApplicationResources.groovy, needed by Resources plugin

Workarounds

  • Running test-app gives: ERROR resource.ResourceMeta - Resource not found: … on some angularjs js files
    • All it's good the same, those files are not required in test phase (at least now)
