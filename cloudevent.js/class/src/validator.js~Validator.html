    Validator

    Generic Validator implementation.

    Note that all methods here are static, so no need to instance this class; see it as an Utility/Companion class.

    Test:

    • ensure the Validator class (direct reference to it) works good
    • ensure some (edge cases for) validation functions are right
    • ensure some (less used) validation functions are right
    • ensure some (utility) functions are right
    • ensure validation functions on standard properties are right
    • ensure validation functions to filter bad object instances raise exceptions
    • ensure validation functions to throw exceptions, works good

    public static getArgumentName(arg: object): string source

    Return the name of the variable given as argument.

    To use it, must be called using the trick to wrap argument inside an object literal, so for example use it inside a template string with: ${getArgumentName({x})} with nested objects (like with options with a property x inside) you need to use with: ${getArgumentName({options})}.${getArgumentName({x})}. Note that this could have been written (in a shorter way) as an arrow function.
