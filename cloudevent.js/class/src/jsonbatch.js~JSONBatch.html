    JSONBatch

    JSONBatch implementation.

    See:

    • https://github.com/cloudevents/spec/blob/master/json-format.md#4-json-batch-format

    Test:

    • ensure serialization functions works good on undefined and null arguments, and even on empty and bad ones
    • ensure isValid and validate works good on undefined and null arguments, and even on empty and bad ones
    • ensure isValid and validate works good on array and related items
    • ensure isValid and validate works good on plain object and even CloudEvent instance and CloudEvent subclasses and not

    public static deserializeEvents(ser: string, options: object): object[] source

    Deserialize/parse the given JSONBatch from JSON format. Note that standard CloudEvent deserialization will be called for any CloudEvent instance, nothing other; so options are the same used in CloudEvent related method.

    See CloudEvent.deserializeEvent.

    Params:

    NameTypeAttributeDescription
    ser string
    • nullable: false

    the serialized JSONBatch to parse/deserialize
    options object
    • optional
    • default: {}

    optional deserialization attributes Additional options valid here: logError (boolean, default false) to log to console deserialization errors throwError (boolean, default false) to throw serialization errors

    Return:

    object[]

    the deserialized batch as a JSONBatch (so a CloudEvent array instance)

    Throw:

    Error

    if ser is undefined or null, or an option is undefined/null/wrong

    Error

    in case of JSON parsing error

    TypeError

    if ser is not a JSONBatch representation
